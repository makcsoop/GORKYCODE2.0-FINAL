# Smartopolis Admin Panel

Админ-панель для платформы Smartopolis (Аналитический центр города).

## Структура проекта

- `index.html` - Главная страница с рейтингом (Мой рейтинг)
- `message.html` - Страница карточки сообщения
- `projects.html` - Страница городских проектов
- `styles.css` - Основные стили
- `script.js` - JavaScript для интерактивности

## Особенности

- Темный сайдбар с навигацией
- Светлый основной контент
- Страница рейтинга с кругом 90%, графиком и отзывами
- Страница карточки сообщения с табами и формой
- Адаптивный дизайн
- Русский язык интерфейса

## Запуск

Просто откройте `index.html` в браузере.
